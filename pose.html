My Project

Tour Package Presentation

This project allowed me to enhance my creativity through visual design and layout, while also improving my public communication skills by presenting information clearly and engagingly. Emblem Expedition Game

Together with my team, we developed a text-based game in C++ that applied programming logic, menu systems, and interactive gameplay design. The project challenged us to structure the code efficiently, handle user inputs, and create branching storylines that kept the gameplay engaging. We implemented features such as decision-based menus, score tracking, and simple mechanics that required players to think critically.

Electric Circuit Design

We built and tested an electronic circuit using a breadboard, with the goal of making the components light up. This project gave us hands-on experience in connecting wires, resistors, and LEDs, while learning how current flows through a simple circuit. It required careful assembly and troubleshooting to ensure the circuit worked properly.
